# Supply-Chain-Analytics-Dashboard
This project focuses on analyzing end-to-end supply chain performance by transforming raw operational data into meaningful insights using Power Query, Data Modeling, and Advanced DAX. It helped me understand how analytics can support better decision-making in areas such as inventory management, logistics efficiency, and vendor performance.

🔍 Key Learnings & Applications<br>
• Cleaned and transformed supply chain datasets using Power Query<br>
• Designed a structured data model for efficient reporting and analys<br>
• Developed advanced DAX measures for KPIs such as Closing Stock, Inventory Turnover, and On-Time Delivery<br>
• Built dynamic calculations for monitoring inventory levels, order status, and production flow<br>
• Implemented interactive slicers and filters to enable flexible data exploration<br>

📊 Dashboard Insights<br>
• Inventory performance and closing stock analysis<br>
• Order and production tracking across the supply chain<br>
• Vendor performance evaluation<br>
• Logistics and delivery efficiency monitoring<br>
• Overall supply chain health and operational KPIs<br>

📈 Interactive Visualizations<br>
• Maps for logistics and distribution analysis<br>
• Charts for stock levels and operational trends<br>
• Dynamic dashboards with filters and slicers for deeper insights<br>

🛠 Tools & Technologies Used<br>
• Power BI Desktop - Main data visualisation platform used for report creation.<br>
• Power Query - Data transformation and cleaning layer for reshaping and preparingh data.<br>
• DAX (Data Analysis Expressions) - Used for calculate measures, dynamic visuals, and conditional logic.<br>
• Data Modeling - Relatinship established among tables.
