# Supply-Chain-Analytics-Dashboard
This project focuses on analyzing end-to-end supply chain performance by transforming raw operational data into meaningful insights using Power Query, Data Modeling, and Advanced DAX. It helped me understand how analytics can support better decision-making in areas such as inventory management, logistics efficiency, and vendor performance.

🔍 Key Learnings & Applications
• Cleaned and transformed supply chain datasets using Power Query
• Designed a structured data model for efficient reporting and analys
• Developed advanced DAX measures for KPIs such as Closing Stock, Inventory Turnover, and On-Time Delivery
• Built dynamic calculations for monitoring inventory levels, order status, and production flow
• Implemented interactive slicers and filters to enable flexible data exploration

📊 Dashboard Insights
• Inventory performance and closing stock analysis
• Order and production tracking across the supply chain
• Vendor performance evaluation
• Logistics and delivery efficiency monitoring
• Overall supply chain health and operational KPIs

📈 Interactive Visualizations
• Maps for logistics and distribution analysis
• Charts for stock levels and operational trends
• Dynamic dashboards with filters and slicers for deeper insights

🛠 Tools & Technologies Used
• Power BI Desktop - Main data visualisation platform used for report creation.
• Power Query - Data transformation and cleaning layer for reshaping and preparingh data.
• DAX (Data Analysis Expressions) - Used for calculate measures, dynamic visuals, and conditional logic.
• Data Modeling - Relatinship established among tables.
